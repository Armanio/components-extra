# Palette

<br/>

You will find on this page all the `colors` from **components-extra**'s default theme.
You can of course override all those on your end. See [this page](/theming/customization) for more details.

## Default Theme

**components-extra**'s default theme is available [here](/theming/customization#default-theme).

**material-ui**'s default theme palette is available [here](https://material-ui.com/customization/palette/).

import ColorBoxes from './docs-utils/ColorBoxes'
import { createTheme, invertTheme } from 'theme'
const defaultTheme = createTheme()
const { palette: lightTheme } = defaultTheme
const { palette: darkTheme } = createTheme(true)
const { palette: invertedTheme } = invertTheme(defaultTheme)

### Light theme (default)

<ColorBoxes palette={lightTheme} />

### Dark theme

<ColorBoxes palette={darkTheme} />

### Inverted main colors

To invert the main colors of the theme (ie `primary` and `secondary`), you can pass the
prop `invertMainColors` to `true` to the `<StyledProvider>`.

<ColorBoxes palette={invertedTheme} />
