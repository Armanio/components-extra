# Palette

<br/>

You will find on this page all the `colors` from **components-extra**'s default theme.
You can of course override all those on your end. See [this page](/theming/customization) for more details.

## Default Theme

**components-extra**'s default theme is available [here](/theming/customization#default-theme).

**material-ui**'s default theme palette is available [here](https://material-ui.com/customization/palette/).

import ColorBox from './docs-utils/ColorBox'
import createTheme, { createInvertedTheme } from 'theme'
const { palette: lightTheme } = createTheme()
const { palette: darkTheme } = createTheme(true)
const { palette: invertedTheme } = createInvertedTheme()
